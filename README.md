# Decode Labs Data Science Internship

## Overview

This repository contains the complete work completed during the Decode Labs Data Science Internship. The internship focused on applying Data Science, Machine Learning, and Business Analytics techniques to solve real-world problems using Python and modern data analysis tools.

The internship was divided into three major projects:

### Week 1 – Advanced EDA & Feature Engineering

Performed exploratory data analysis on an E-Commerce dataset to uncover trends, handle missing values, detect outliers, engineer new features, and generate business insights.

### Week 2 – Fraud Detection Pipeline

Built a complete machine learning pipeline for fraud detection including data preprocessing, class imbalance handling using SMOTE, model training, evaluation, and performance comparison.

### Week 3 – Customer Segmentation

Applied unsupervised learning techniques including PCA and K-Means Clustering to identify customer segments and create business-oriented customer personas.
